# Belajar Huruf & Kata

Project sederhana: permainan edukasi untuk menyusun huruf membentuk kata.

Teknologi:
- React + Vite
- Tailwind CSS

Cara menjalankan:
1. Install dependensi: `npm install`
2. Jalankan development server: `npm run dev`
3. Buka `http://localhost:5173`

File penting:
- `src/components/WordGame.jsx` : logic game
- `src/App.jsx` : layout utama
